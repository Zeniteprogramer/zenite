ZNT Language — Manual Oficial

Visão Geral
ZNT é uma linguagem de programação criada para ser simples, rápida de aprender e fácil de entender.
Ela foi feita com foco em leitura humana, usando comandos curtos, intuitivos e organizados, sem excesso de símbolos complicados.
Enquanto muitas linguagens focam apenas em executar comandos, o ZNT foi pensado para controlar a lógica de forma clara e inteligente.

O que é o ZNT?
Se compararmos com outras tecnologias:

HTML = o esqueleto → estrutura
CSS = a pele → aparência
JavaScript = os movimentos → ações e interações
ZNT = o cérebro → lógica, decisões e controle

ZNT é o cérebro do sistema
Ele decide:

o que acontece
quando acontece
como acontece
por que acontece

O ZNT organiza o pensamento do sistema.
Ele não é apenas uma linguagem de programação.
Ele é o núcleo da execução.

Onde o ZNT pode ser usado?
ZNT pode ser usado para:

criação de sistemas
automação de tarefas
jogos e lógica de gameplay
inteligência de processos
scripts rápidos
bots e sistemas automáticos
programas desktop
projetos experimentais
aprendizado de programação
controle de dados e decisões

Exemplo:


HTML constrói a casa


CSS pinta a casa


JavaScript abre as portas


ZNT decide quem pode entrar

Estrutura Básica
Todo código funciona com:
keyword + ação + valor
Exemplo:
show "Olá mundo"


Variáveis
let
Guarda valores normais.
let nome = "Rin"
mark
Valor importante.
mark level = 99
lock
Valor travado (não muda).
lock game = "ZNT"
core
Valor principal/global.
core app = "Zenite"
flux
Altera valor depois.
flux level = 100


Entrada e Saída
show
Mostra algo na tela.
show "Hello"
echo
Retorna ou repete valor.
echo nome
scan
Recebe entrada do usuário.
let nome = scan


Condições
check
Primeira verificação.
check idade >= 18    show "Maior de idade"
gate
Segunda condição.
gate idade >= 16    show "Quase adulto"
other
Caso contrário.
other    show "Menor de idade"

Repetição
loop
Repete quantidade definida.
loop 5    show "Executando"
spin
Loop infinito.
spin    show "∞"
stop
Para execução.
stop
skip
Pula etapa.
skip


Funções
forge
Cria função.
forge hello    show "Olá"
call
Executa função.
call hello
back
Retorna valor.
back resultado

Texto
join
Junta textos.
join "Olá" " Mundo"
cut
Divide textos.
cut "a,b,c" ","
find
Procura algo.
find "player"


Sistema
wake
Inicia processo.
wake sistema
sleep
Pausa execução.
sleep 5
send
Envia dados.
send dados
pull
Importa ou puxa dados.
pull sistema


Extras Exclusivos do ZNT
spark
Inicia algo importante.
spark engine
shade
Oculta informação.
shade senha
bloom
Cria crescimento.
bloom projeto
drift
Move dados.
drift save to backup
zen
Modo seguro.
zen system


Comentários
# comentário normal
## comentário importante


Regras Gerais


ZNT respeita a ordem de execução


comandos sempre vêm antes dos valores


strings usam aspas


blocos usam indentação


nomes devem ser simples e claros
