ZNT Language — Manual Oficial


🧠 ZNT Language — Manual
🌟 O que é o ZNT?

ZNT é uma linguagem de programação feita pra ser:

👉 fácil de ler
👉 fácil de entender
👉 tipo “frases humanas” dizendo o que o programa faz

🧩 Pra que serve?

O ZNT serve pra mandar o computador fazer coisas:

tomar decisões 🤔
repetir tarefas 🔁
guardar informações 📦
criar sistemas e jogos 🎮
🧠 Ideia principal

Pensa assim:

HTML = estrutura da casa
CSS = pintura da casa
JS = movimentos
ZNT = quem manda em tudo (o cérebro) 🧠
✍️ Como o código funciona

Tudo no ZNT é assim:

comando + valor

Exemplo:

show "Olá mundo"

👉 isso significa: mostrar na tela

📦 VARIÁVEIS (guardar coisas)
let nome = "Rin"

👉 cria uma informação simples

mark level = 99

👉 coisa importante

lock game = "ZNT"

👉 não pode mudar

flux level = 100

👉 pode mudar depois

👀 MOSTRAR E LER
show "Oi"

👉 mostra na tela

echo nome

👉 repete um valor

let nome = scan

👉 usuário digita algo

🤔 DECISÕES (IF / ELSE)
if idade == 18
    show "Maior de idade"

👉 se for verdade, faz isso

else if idade == 16
    show "Quase adulto"

👉 outra condição

else
    show "Menor de idade"

👉 se nada for verdade

🔁 REPETIÇÃO
loop 5
    show "Executando"

👉 repete 5 vezes

spin
    show "∞"

👉 repete sem parar

stop

👉 para tudo

⚙️ FUNÇÕES
forge hello
    show "Olá"

👉 cria uma função

call hello

👉 usa a função

back resultado

👉 devolve algo

📁 SISTEMA
wake sistema

👉 liga o sistema

sleep 5

👉 espera

send dados

👉 envia algo

pull sistema

👉 puxa dados

🧪 EXEMPLO SIMPLES COMPLETO
let nome = "Rin"
mark level = 99

if level >= 50
    show "Forte"
else
    show "Iniciante"

forge hello
    show "Bem-vindo"

call hello
