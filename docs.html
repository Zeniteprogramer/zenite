🧠 ZNT Language — Manual
🌟 O que é o ZNT?

ZNT é uma linguagem de programação feita pra ser:
👉 fácil de ler
👉 fácil de entender
👉 tipo “frases humanas” dizendo o que o programa faz

🧩 Pra que serve?
O ZNT serve pra mandar o computador fazer coisas:
tomar decisões 🤔
repetir tarefas 🔁
guardar informações 📦
criar sistemas e jogos 🎮

🧠 Ideia principal
Pensa assim:
HTML = estrutura da casa
CSS = pintura da casa
JS = movimentos
ZNT = quem manda em tudo (o cérebro) 🧠

✍️ Como o código funciona
Tudo no ZNT é assim:
comando + valor

Exemplo:
show "Olá mundo"
👉 isso significa: mostrar na tela

📦 VARIÁVEIS (guardar coisas)
let nome = "Rin"
mark level = 99
lock game = "ZNT"
flux level = 100

👀 MOSTRAR E LER
show "Oi"
echo nome
let nome = scan

🤔 DECISÕES (IF / ELSE)
if idade == 18
    show "Maior de idade"
else if idade == 16
    show "Quase adulto"
else
    show "Menor de idade"

🔁 REPETIÇÃO
loop 5
    show "Executando"
spin
    show "∞"
stop

⚙️ FUNÇÕES
forge hello
    show "Olá"
call hello
back resultado

📁 SISTEMA
wake sistema
sleep 5
send dados
pull sistema

🧪 EXEMPLO SIMPLES COMPLETO
let nome = "Rin"
mark level = 99

if level >= 50
    show "Forte"
else
    show "Iniciante"

    show "Bem-vindo"
